# Static Site Vercel Starter

Generated by `stripe build` as a lightweight static-site starting point.

## What this starter gives you

- Next.js App Router starter with a polished landing page
- Tailwind CSS 4 styling
- A built-in Vercel deployment script wired to the same pulled project credentials used by the other Vercel templates here
- A simple AI handoff prompt in `prompts/starter-to-product.md`


## Local development

1. Install dependencies:
   `npm install`
1. Start the app:
   `npm run dev`
1. Edit the landing page in `app/page.tsx`
1. Update metadata in `app/layout.tsx`
1. Deploy to Vercel with:
   `npm run deploy`

`npm run deploy` expects the pulled Vercel env vars to be available: `VERCEL_TOKEN`, `VERCEL_PROJECT_ID`, `VERCEL_TEAM_ID`, `VERCEL_ORG_ID`, `VERCEL_PROJECT_URL`, `VERCEL_URL`.

## Customize with AI

Use one of these commands from the repo root:

- `codex "Help me turn this into a real product. Follow prompts/starter-to-product.md."`
- `claude "Help me turn this into a real product. Follow prompts/starter-to-product.md."`
- `agent "Help me turn this into a real product. Follow prompts/starter-to-product.md."`

If you are using another AI coding tool, open `prompts/starter-to-product.md` and paste it into the assistant.
